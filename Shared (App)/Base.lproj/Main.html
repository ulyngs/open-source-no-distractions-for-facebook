This Safari extension hides distractions on Facebook, and lets you toggle them on when you need them.
It's open source and collects zero data - you can find the source code on GitHub.

To turn it on, open Safari and go to facebook.com. Then:
1) In the toolbar tap 'Aa' -> 'Manage Extensions', and toggle the extension on.
2) Tap the extension and allow it to access facebook.com
